# Waveshare ESP32-P4-WIFI6-Touch-LCD-3.5 Product Engineering Sample Program

This product is a dual-core plus single-core RISC-V high-performance development board based on ESP32-P4 chip. It is equipped with a 3.5-inch 320 × 480 resolution IPS touch display. Support rich human-computer interaction interface
